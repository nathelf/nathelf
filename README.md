## 📚 Projetos em Destaque

### 🧠 Geração Automática de Documentação com LLMs
Pesquisa e implementação de pipeline para geração automática de documentação técnica a partir de código-fonte e especificações de APIs, utilizando modelos como Zephyr-7B, FLAN-T5 e LLaMA.

Inclui:
- Engenharia de prompts
- Estruturação automática em Markdown
- Conversão para Word e PDF
- Avaliação qualitativa dos resultados

---

### 🌱 Classificação de Imagens Agrícolas com Vision Transformer (ViT)
Modelo baseado em Vision Transformer para classificação de imagens (solo, palha e planta).

O projeto inclui:
- Pré-processamento de imagens
- Treinamento e validação com PyTorch
- Interface gráfica para classificação
- Organização automática de imagens por categoria
